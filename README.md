# Python Task Manager

## Overview

A feature-rich desktop task management application built with Python and Flet. Manage your daily tasks with priorities, scheduling, and productivity tracking.

## Features

- **Task Creation**: Add tasks with titles, descriptions, and due dates
- **Priority Levels**: Organize by High, Medium, or Low priority
- **Categories**: Group tasks by project or category
- **Data Persistence**: All tasks saved locally
- **Clean UI**: Modern interface built with Flet framework
- **Cross-Platform**: Works on Windows, Mac, and Linux

## Installation

### Prerequisites
- Python 3.7 or higher
- pip (Python package manager)

### Setup

1. Download this repository (click green "Code" button ‚Üí "Download ZIP")
2. Extract the ZIP file
3. Open Terminal/Command Prompt in that folder
4. Install dependencies:
```bash
   pip install -r requirements.txt
```
5. Run the app:
```bash
   python main.py
```

## How to Use

1. Click "Add Task" to create new tasks
2. Set priority levels and due dates
3. Check off completed tasks
4. Delete or edit tasks as needed
